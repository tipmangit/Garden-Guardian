Project Title: Garden-Guardian - Wireless Feline Monitoring and Control System via Fluid Actuation.

Description: This application is intended to reduce the amount of stray cats cluttering their mess in your homes. 
This works with a camera sensor that detects if there are nearby cats in the area, and then if there is at least one detected then there 
would be a notification through the app wherein you can get notified that there is an anomaly detected within the area.

Technologies Used: Flutter, Dart, IoT, Cloud Computing (mention the cloud platform you’re using).

Features: 
-Intelligent Cat Detection System
-Smart Pan-Tilt Camera Control
-Automatic Tracking
-Manual Control Interface
-Instant Push Notifications & Alerts
-Remote Sprinkler Activation
-Image Capture & Storage
-Comprehensive Event Logs
-"Known Cat" Ignore Function
-User Settings & Customization
